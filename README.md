### Features
- BVH acceleration
- OBJ triangle mesh loading
- Depth of field
- Texture mapping
- Antialiasing
- Parallel processing
- Primitives
  - Sphere
  - Triangle
  - Ground
- Materials
  - Lambertian
  - Metal
  - Dielectric
  - Diffuse light
  
### Installation
```
rake install
```
